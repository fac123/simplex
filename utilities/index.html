What are utility classes?

Simplex provides a series of simple utility classes that make it easy to construct complex user interfaces. Utility classes allow authors to apply styles directly to HTML elements without having to write additional CSS.

Utility classes include: color, background-color, font-sizes, hidden and visible, floats, centering, text-alignment, margins and padding.
