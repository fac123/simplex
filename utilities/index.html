Utility classes

Utility classes make it easy to construct complex user interfaces. These classes apply styles directly to HTML elements without having to write additional CSS.

Utility classes have the following prefixes:

.c-
Color utility classes
.bg-
Background-color utility classes
.u-
All other utility classes
