Message

These utility classes allow authors to style hint and error messages.

Usage

Used to apply error or hint classes to messages relating to form controls.

Accessibility

These messages must be programatically associated with their relevant form control.

.message-error

message-error 
<span class="message-error">message-error</span>

.message-hint

message-hint 
<span class="message-hint">message-hint</span>
