Font-size utility classes

Purpose

Font-size utility classes allow authors to apply a specific font-size to any element and at different screen sizes.

Usage

Font-size utilities can be used to override existing element font-sizes and create unique layouts.

Accessibility

The smallest font-sizes should be avoided except for specific circumstances as they are hard to read for many audiences.
