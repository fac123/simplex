Font-size

Font-size utility classes allow authors to apply a specific font-size to any element and at different screen sizes.

Usage

Used to apply specific font sizes to any element or component.

Accessibility

The smallest font-sizes (.font10, .font12) should be avoided except for specific circumstances as they are hard to read for many audiences.
