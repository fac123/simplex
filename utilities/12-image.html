Image

These utility classes allow authors to style images. The .u-image--left class floats the image to the left. The .u-image--right class floats the image to the right.

Overall usage

Used to apply positioning to images.

Accessibility

  • The alt attributes must be used with all images.
  • The alt attribute must be descriptive except for decorative images, when the value can be null.
