Hidden utility classes

Purpose

Hidden utility classes allow authors to hide content at specific screen sizes.

Usage

Hidden utility classes can be used to shorten instructions or button texts at smaller screen sizes where there is less screen space. This can be used in conjunction with visible utility classes.

Accessibility

Hiding content can present issues for assistive technologies so these utility classes should be used carefully. Meaningful content should never be completely removed.

Example
Hidden at all sizes
Hidden at xs only
Hidden at sm only
Hidden at md only
Hidden at lg only
Markup
<div class="hidden">Hidden at all sizes</div>
<div class="hidden-xs">Hidden at xs only</div>
<div class="hidden-sm">Hidden at sm only</div>
<div class="hidden-md">Hidden at md only</div>
<div class="hidden-lg">Hidden at lg only</div>
