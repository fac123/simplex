Float utility classes

Purpose

Float utility classes allow authors to move elements to the left and right at specific screen sizes.

Usage

Float utility classes can be used to make elements sit beside other elements, and then have then sit above/below at smaller screen sizes.

Example

All sizes

float-left
float-right
float-none

xs only

float-left-xs
float-right-xs
float-none-xs

sm only

float-left-sm
float-right-sm
float-none-sm

md only

float-left-md
float-right-md
float-none-md

lg only

float-left-lg
float-right-lg
float-none-lg
