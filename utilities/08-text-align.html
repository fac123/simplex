Text-align utility classes

Purpose

Text-align utility classes allow authors to align content to the left, right or center at specific screen sizes.

Usage

Self-explanatory.

Accessibility

N/A

Example

All sizes

.text-left
.text-right
.text-center

xs only

.text-left-xs
.text-right-xs
.text-center-xs

sm only

.text-left-sm
.text-right-sm
.text-center-sm

md only

.text-left-md
.text-right-md
.text-center-md

lg only

.text-left-lg
.text-right-lg
.text-center-lg
