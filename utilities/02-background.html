Background-color utility classes

Purpose

Background-color utility classes allow authors to apply colors to any element, including extending components.

Usage

Tints of the primary and secondary color can be used as background for feature panels or colored rows.

Accessibility

Color combinations should only be used as presented to maintain color contrast ratios.
