Checkbox

The checkbox component represents an <input> element with a type of checkbox and its explicitly associated <label> element.

Usage

Used to present one or more single or multiple choice options to users.

Accessibility

The <input> element needs an explicitly associated <label>. The contents of the <label> must come after the <input> element.

Basic checkbox

<div class="input-checkbox">
  <label class="input-checkbox__label" for="example1">
    <input class="input-checkbox__control" type="checkbox" name="example3" id="example1">
    <span class="input-checkbox__content">Checkbox</span>
  </label>
</div>

Required checkbox

<div class="input-checkbox input-checkbox--required">
  <label class="input-checkbox__label" for="example2">
    <input class="input-checkbox__control" type="checkbox" name="example3" id="example2" required>
    <span class="input-checkbox__content">Checkbox label <span class="required">(Required)</span></span>
  </label>
</div>

Invalid checkbox

<div class="input-checkbox isInvalid">
  <label class="input-checkbox__label" for="example3">
    <input class="input-checkbox__control" type="checkbox" name="example3" id="example3" required>
    <span class="input-checkbox__content">Checkbox label <span class="required">(Required)</span></span>
  </label>
</div>

Disabled checkbox

<div class="input-checkbox input-checkbox--disabled">
  <label class="input-checkbox__label" for="example4">
