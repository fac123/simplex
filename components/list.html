List

The list component represents variations on the unordered list.

Usage

  • The list-unstyled variation can be used when a list requires no bullets
  • The list-inline variation can be used where a list is required to be inline
  • The list-inline-greater variation can be used where an inline list requires a "greater-than" symbol
  • The list-inline-pipe variation can be used where an inline list requires a "pipe" symbol
  • The list-inline-slash variation can be used where an inline list requires a "slash" symbol
  • The list-pagination variation can be used for pagination
  • The list-navigation variation can be used for page navigation
