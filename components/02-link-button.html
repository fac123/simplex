link-button

The link-button component represents a clickable link that has been styled to look like a button.

Usage
  • The link-button component should only be used when the purpose is to send the user to a new URL.
  • If the aim is to trigger a form submission or to trigger an event, the button component should be used.
Accessibility
  • There must be sufficient contrast between background and foreground colors
  • The content of the link should be meaningful
  • The static, focus, hover and active states of the button should be defined
  • There must be a clearly visible focus state, and the method should be consistent across all focusable elements.
